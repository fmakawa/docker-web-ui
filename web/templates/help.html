FAQ:


Can I make my own run command and run it in the UI?

No. We feel that the CLI is best for this and are currently not looking into enabling this feature

Can we rerun deleted containers?

No, once deleted you must run them through the CLI.

It should be noted that all functions to create new containers, images, configs, volumes, networks .etc have not been enabled. You can only manipulate already existing assets. All creation should be done via the CLI
